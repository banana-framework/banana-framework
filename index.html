banana.utils.version.get_version(version=None)[source]¶

Returns semantic version representation (PEP-440)

Parameters:version (VersionInfo) – structured version. By default banana.VERSION
Returns:semantic version representation
Return type:str
Examples:
>>> from banana.utils.version import VersionInfo, get_version
>>> version = VersionInfo(0, 1, 2, 'beta', 10)
>>> get_version(version)
'0.1.2b10'

>>> from banana.utils.version import VersionInfo, get_version
>>> version = VersionInfo(0, 1, 2, 'final', 10)
>>> get_version(version)
'0.1.2'

>>> from banana import VERSION
>>> from banana.utils.version import VersionInfo, get_version
>>> get_version() == get_version(VERSION)
True
